利用規約

第1条（定義）

  • 「本アプリ」とは、個人開発者（以下「開発者」といいます。）が提供するスマートフォンアプリ「あなたの歌詞が」（以下「本アプリ」といいます）を意味します。
  • 「利用者」とは、本アプリを利用する者を意味します。

第2条（利用規約の適用）

利用者は、本アプリの利用に際し、本規約を遵守するものとします。

第3条（利用者の権利と義務）

利用者は、本アプリを、本規約に従って利用することができます。 利用者は、本アプリの利用にあたり、以下の行為をしてはなりません。

  • 本アプリの著作権、商標権その他の知的財産権を侵害する行為
  • 本アプリを不正に利用する行為
  • 本アプリの利用にあたり、第三者の権利を侵害する行為
  • 本アプリの運営を妨害する行為
  • 本アプリに不当な情報を掲載する行為
  • 本アプリを利用して犯罪に当たる行為
  • 本規約に違反する一切の行為

第4条（禁止行為）

利用者は、本アプリの利用にあたり、以下の行為をしてはなりません。

  • 本アプリを逆アセンブル、逆コンパイル、リバースエンジニアリング、その他本アプリのソースコード、構造、アイデア等を解析するような行為
  • 本アプリを複製、送信、譲渡、貸与、翻訳、翻案、改変、他のソフトウェアと結合等する行為
  • 本アプリの利用にあたり、虚偽の情報を提供する行為
  • 本アプリの利用にあたり、他人の個人情報やプライバシーを侵害する行為
  • 本アプリの利用にあたり、有害な情報を発信する行為
  • 本アプリの利用にあたり、本アプリの運営者または第三者に損害を与える行為

第5条（損害賠償）

利用者が本規約に違反したことにより、開発者または第三者に損害を与えた場合、利用者は、その損害の賠償を請求される場合があります。

第6条（規約の変更）

開発者は、本規約を変更することがあります。変更後の規約は、開発者のウェブサイトに掲載した日から効力を生じます。

第7条（準拠法）

本規約は、日本法に準拠します。
